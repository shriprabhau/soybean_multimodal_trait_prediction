# Soybean_multimodal_trait_prediction

The study uses three datasets, SNPs, Gene PAVs and Protein family PAV, along with a location covariate. 

Each trait folder contains the notebooks for the singular models, ensemble models and multi-input neural networks.

SNP datasets are encoded as 0 = homozygous reference alleles, 1 = heterozygous alleles and 2 = homozygous alternate alleles

PAV datasets have binary encoding with 1 = presence and 0 = absence
